Try ACL2

ACL2 is:

  • A Compuational Logic: A complex mechanized theorem prover that can reason about code written in...
  • Applicitave Common Lisp: A significantly restricted subset of Common Lisp that doesn't allow for variables to be modified in place.

Some things to try (click to insert):

  • (+ 1 1)
  • (append '(1 2 3) '(4 5 6))
  • (thm (= (+ a b) (+ b a)))
  • (defun fact (n) (if (zp n) 1 (* n (fact (- n 1)))))
  • (defun sum (xs) (if (endp xs) 0 (+ (first xs) (sum (rest xs)))))
  • (thm (= (sum (append xs ys)) (+ (sum xs) (sum ys))))
